  • 여러가지 경사하강법

  1. 정규 방정식과 유사 역행렬

    - 정규방정식(Normal equation)
    : 식으로 표현하면 다음과 같다.
    θ = (X^T * X)^(-1) * X^T * y
    여기서, θ : 가중치 매개변수 벡터(회귀 계수), X : 입력 행렬 (샘플 x특성의 크기) y: 예측하려는 목표 변수의 벡터, X^T: X의 전치 행렬, (X^T * X)^(-1): 역행렬을 구한 결과를 사용하여 회귀 계수를 구하여 그를 활용하여 일차방정식의 형태로 표현하는 것이다.

    - 유사 역행렬(pseudo-inverse)
    : 유사 역행렬의 개념을 사용하여 본래 정규방정식에서 입력 행렬의 역행렬이 존재하지 않을때 계산할 수 있게 한다.

  2. 배치 경사 하강법

    - 배치 경사 하강법(Batch Gradient Descent, BGD)
    : 알고리즘의 목표인 손실 함수를 최적화 시키는 것이기 때문에 최적의 가중치를 찾아야 하는데 가중치를 학습률과 그 전 가중치를 이용하여 구하는 아이디어이다. 식은 다음과 같다.
    w_{n} = w_{n-1} - α∇f(w_{n-1})
    - 경사 하강법의 손실 함수의 기울기 계산에 전체 학습 데이터셋의 크기와 동일하게 잡는 방법으로 경사 하강법의 대상이 배치 크기와 동일하다는 것이다. 단점으로 데이터셋 모두를 대상으로 하므로 파라미터가 한번 이동할 때마다 계산해야 할 값이 지나치게 많아 시간복잡도와 공갑복잡도 둘 모두 높다.
    - 특징은 모든 데이터를 대상으로 경사 하강법을 실시하므로 안정적으로 수렴한다. 그러므로 수렴까지 발생하는 파라미터의 수는 적다. 안정적으로 움직이기 때문에 local minimum문제에 빠질 수 있다. 학습 데이터셋이 커지면 배치 크기도 커지기 때문에 리소스 소모가 매우 크다.

  3. 확률적 배치 경사 하강법

    - 확률적 배치 경사 하강법(Stochastic Gradient Descent, SGD)
    : 학습 데이터셋에서 무작위로 한 개의 샘플 데이터셋을 추출하고, 그 샘플에 대해서만 배치 경사 하강법를 계산하는 방법이다.
    - 확률적 경사 하강법은 확률적 알고리즘을 사용한 방법이다. 확률적 알고리즘의 예로는 밀러-라빈 소수판별 알고리즘이 있다.
    - 전체 학습 데이터셋을 대상으로 하고 학습하는 것과 local minimum에 빠지기 쉬운 것을 개선하기 위해 만들어진 것이다
    - 학습 데이터셋에서 무작위로 한 개의 샘플 데이터셋을 추출하고, 그 샘플에 대해서만 기울기를 계산하는 방법이다.
    - 샘플 데이터셋에 대해서만 기울기(Gradient)를 계산하므로, 매 반복에서 다뤄야 할 데이터 수가 적어 학습 속도가 빠르다.
    - 하나의 샘플로만 계산하기에 공강복잡도가 낮고, 매우 큰 데이터셋 또한 배치 경사 하강법보다 훨씬 쉽게 학습 가능하다.
    - 무작위로 뽑아낸 샘플에 대해서만 경사를 구하므로 배치 경사 하강법보다 불안정하게 움직인다.
    - global minimum에 정확히 도달하지 못 할 가능성이 있다. 하지만, 이 경사하강법은 불안정하기 때문에 local minimum에 빠진 다 할 지라도 쉽게 빠져 나올 수가 있어 global minimum에 더 근접한다.

  4. 미니 배치 경사 하강법

    - 미니 배치 경사 하강법(mini Batch gradient descent, SGD)
    : 미니 배치 경사 하강법은 BGD와 SGD의 장점을 조합하여 만든 것으로 미니 배치를 생성하여 그 미니 배치에 배치 경사 하강법으로 계산하는 방법이다.
    - 미니 배치 크기를 줄였기 때문에 파라미터 수가 확률적 경사 하강법보다 줄어들고 안정적으로 변한다. 그러므로 빠르게 최적해에 가까이 갈 수 있지만, local minimum이 확률적 경사하강법보다 상대적으로 빠지기 쉽다.
    - 미니 배치는 학습 데이터셋의 크기를 배치 크기만큼 나누었을 때, 딱 떨어지는 크기가 좋은 학습 데이터로 사용될 수 있다, 왜냐하면, 남은 몇 개가 있다면 그 데이터를 과평가할 수 있기 때문이다.
